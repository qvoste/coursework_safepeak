# SafePeak
Веб-приложение страховой компании — оформление полисов по здоровью, имуществу, авто, жизни и путешествиям.

## Стек
- PHP для бэкэнда
- HTML + СSS + JS для фронтенда
- json для хранения данных

## Архитектура

```
safepeak/
│
├── index.html                     ← Главная страница
├── insurance.html                 ← Страница страхования
├── profile.html                   ← Личный кабинет
├── .htaccess                      ← Безопасность Apache
│
├── api/
│   ├── config.php                 ← Константы, типы страхований, настройки сессий
│   ├── auth.php                   ← Регистрация, вход, выход, проверка сессии
│   ├── policies.php               ← Эндпоинт оформления и получения полисов
│   ├── PolicyModel.php            ← Модель полиса
│   └── UserModel.php              ← Модель пользователя
│
├── assets/
│   ├── css/
│   │   ├── style.css              ← Общие стили, навигация, модалки, toast
│   │   ├── home.css               ← Стили главной страницы
│   │   ├── insurance.css          ← Стили страницы страхования
│   │   └── profile.css            ← Стили личного кабинета
│   └── js/
│       ├── api.js                 ← Все AJAX-запросы к бэкенду
│       └── ui.js                  ← Интерфейсная логика (Toast, AuthModal, AuthState и др.)
│
└── data/                          ← JSON-хранилище (закрыто через .htaccess)
    ├── users.json
    └── policies.json
```

## Разделение ответственностей

| Слой | Что делает |
|---|---|
| PHP (`api/`) | Валидация, аутентификация, CSRF, запросы к модели |
| HTML (`.html`) | Разметка, структура страниц |
| CSS (`assets/css/`) | Оформление, анимации, адаптивность |
| JS (`assets/js/`) | Интерактивность, AJAX, DOM-анимации |

## Безопасность

| Механизм | Реализация |
|---|---|
| Хеширование паролей | `password_hash($pw, PASSWORD_BCRYPT, ['cost' => 12])` |
| XSS | `htmlspecialchars(..., ENT_QUOTES, 'UTF-8')` на каждый вывод; `sanitize()` на входе |
| Cookie | `httponly=1`, `use_strict_mode=1` |
| Закрытый JSON | `.htaccess` блокирует прямой доступ к `data/*.json` |
| Валидация | `filter_var`, `preg_match`, `array_key_exists` для `type` полиса |
